README FOR TRANSPORTIT WEB APPLICATION	

To see the web applications full functionality you have to:

1. Run the code (from terminal run- npm start) in Chrome browser
2. Create a new account with valid email to receive the confirmation email.
3. Click on "About Us" on the navigation bar and read the idea section to understand how the application works.
4. Create a sender with attributes from STARLAND to PARADISE, dates from today until the end of August and minimal dimensions.
5. Watch the video on by clicking the youtube link on the footer section to see an example of this application.
